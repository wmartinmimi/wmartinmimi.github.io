browserlauncher4j

Description

A simple java library for launching a default browser.

Licensed under MIT License.

Example code

package com.example;

import io.wmartinmimi.github.browserlauncher4j.BrowserLauncher;
import java.net.URI;
import java.net.URISyntaxException;

public class BrowserLauncherTest {
    public static void main(String[] args) throws URISyntaxException {
        BrowserLauncher.launch(new URI("https://example.com"));
    }
}

For more example see BrowserLaunchTest.
